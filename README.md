## English <a name="english"></a> 🇺🇸

### 📝 About This Project
This repository serves as a centralized place for my solutions to various [LeetCode](https://leetcode.com/) problems, all implemented using **TypeScript**. The main focus is to:
- 🧠 Sharpen problem-solving skills.
- 📐 Master Data Structures and Algorithms (DSA).
- ✨ Write clean, typed, and efficient code.

### 🚀 Tech Stack
- **Language**: TypeScript `^6.0.0`
- **Compiler/Runtime**: `ts-node`
- **Environment**: Node.js

### 📂 Repository Structure
Solutions are stored in the root directory and follow a clear naming convention:
`{problem-slug}-{id}.ts`

> Example: [longest-common-prefix-14.ts](longest-common-prefix-14.ts)

### 💻 How to Run
1. **Clone the repo:**
   ```bash
   git clone https://github.com/alvarobianor/leetcodes.git
   cd leetcodes
   ```
2. **Install dependencies:**
   ```bash
   yarn
   ```
3. **Run a specific solution:**
   ```bash
   yarn ts-node <filename>.ts
   ```
